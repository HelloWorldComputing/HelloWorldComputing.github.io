How To Build Your Website From Nothing For FREE

First of all, you must have your Google account.

You can create one HERE.






Then, you can go to Github to create your account and set up your page for FREE.


Reference: https://pages.github.com/



1. Using your account to create a repository

# Remember the name of repository name should follow the format:

 username.github.io

where username should be the same as your GitHub username.


Example: my username is HelloWorldComputing, then the name of my repository would be

HelloWorldComputing.github.io





2. Download GitHub Desktop

Download link: https://desktop.github.com/





3. Clone your repository into your GitHub Desktop

A local GitHub file is created.





4. Create your index.html

We know that the first page of the website is the index.html, therefore, we have to create a html file by
any text editor.

Learn more from W3School: https://www.w3schools.com/html/

or

your can use the following code and save the file as index.html

 
<!DOCTYPE html>
  <html>
  <body>
  <h1>Hello World</h1>
  <p>I'm hosted with GitHub Pages.</p>
  </body>
  </html>




5. Place your index.html into local GitHub file

 Just copy and paste your html file created at step 4 into the local GitHub file which created in step 3





6. Update









In addition, you need a AdSense account for earning money.


Your can use your page which was created using GitHub to apply AdSense. Remember to paste the ads code to your GitHub page's html file.


Lastly, WAIT for Google's approval. Done!
